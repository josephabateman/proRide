ProRide — Website Editor

One-time setup

To let this editor save changes to your website, paste a GitHub access token. It's stored only in this browser on this device.

  1. Open GitHub → fine-grained token (log in if asked).
  2. Set Repository access → Only select repositories → josephabateman/proRide.
  3. Under Permissions → Repository permissions, set Contents to Read and write.
  4. Click Generate token, copy it, and paste below.

Tip: don't do this on a shared/public computer.

Make your changes, then publish.
